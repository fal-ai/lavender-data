This guide will walk you through the essential steps to set up the server, create your first dataset and iterate over it.

<Steps>
1. Install the `lavender-data`.

    ```bash
    pip install lavender-data
    ```

2. Start the lavender-data server.

    ```bash
    lavender-data server start --init
    ```
    ```
    lavender-data is running on 0.0.0.0:8000
    UI is running on http://localhost:3000
    API key created: la-...
    ```

    <LinkCard title="Learn more about Server Configuration" href="/server/configuration" />

3. Create the dataset.

    <Tabs syncKey="ui-or-py">
      <TabItem value="cli" label="CLI">


    ```bash
    lavender-data client \
      --api-url http://0.0.0.0:8000 --api-key la-... \
      datasets create \
      --name my_dataset \  # Identifier for the dataset
      --uid-column-name uid  # Column name which can be used as a unique identifier for each row
    ```
    ```json
    {
      "id": "ds-...",
      "name": "my_dataset",
      "uid_column_name": "uid"
    }
    ```

      </TabItem>
      <TabItem value="python" label="Python">


    ```python
    import lavender_data.client as lavender

    lavender.init(api_url="http://0.0.0.0:8000", api_key="la-...")

    dataset = lavender.api.create_dataset(name="my-dataset", uid_column_name="uid")
    ```

      </TabItem>
      <TabItem value="web-ui" label="Web UI">

    Navigate to the [datasets page](http://localhost:3000/datasets) in your browser and click on the "+ Dataset" button.

    ![Datasets Empty](/web-ui/web-ui-datasets-empty.png)

    Enter the dataset name and the unique identifier column name.

    ![Create a Dataset](/web-ui/web-ui-create-dataset.png)

      </TabItem>
    </Tabs>


4. Create the shardset.

    Each shardset can be considered as a directory containing multiple shard files.
    The directory should be flat, containing the shard files only.
    If the order matters, sort shards by the filename.

    <FileTree>
    - /path/to/your/shardset/
      - shard.00000.csv
      - shard.00001.csv
      - ...
    </FileTree>

    <Tabs syncKey="ui-or-py">
      <TabItem value="cli" label="CLI">

    ```bash
    lavender-data client \
      --api-url http://0.0.0.0:8000 --api-key la-... \
      shardsets create \
      --dataset-id ds-... \
      --location file:///path/to/your/shardset  # Where shard files are located (can be `file://`, `s3://`, etc.)
    ```
    ```json
    {
      "id": "ss-...",
      "dataset_id": "ds-...",
      "location": "file:///path/to/your/shardset",
      "shard_count": 0,
      "total_samples": 0
    }
    ```

      </TabItem>
      <TabItem value="python" label="Python">

    ```python
    shardset = lavender.api.create_shardset(
        dataset_id="ds-...",
        location="file:///path/to/the/shardset/",
    )
    ```
      </TabItem>
      <TabItem value="web-ui" label="Web UI">

    Click on "+ Shardset" button in the dataset detail page.
    Enter the path to the directory containing the shard files.
    If you don't have any shards yet, you need to specify the columns manually.

    ![Create Shardset](/web-ui/web-ui-create-shardset.png)

      </TabItem>
    </Tabs>

    <LinkCard title="Learn more about Data Structure" href="/dataset/structure" />

5. Check the shardset and wait until it's synced to the server.


    <Tabs syncKey="ui-or-py">
      <TabItem value="cli" label="CLI">

    ```bash
    lavender-data client \
      --api-url http://0.0.0.0:8000 --api-key la-... \
      shardsets get \
      --dataset-id ds-... --shardset-id ss-...
    ```
    ```json
    {
      "id": "ss-...",
      "dataset_id": "ds-...",
      "location": "file:///path/to/your/shardset",
      "shard_count": 10,
      "total_samples": 100
    }
    ```

      </TabItem>
      <TabItem value="python" label="Python">

    ```python
    shardset = lavender.api.get_shardset(
        dataset_id="ds-...",
        shardset_id="ss-...",
    )
    ```
      </TabItem>
      <TabItem value="web-ui" label="Web UI">

    ![Shardset Created](/web-ui/web-ui-dataset-detail.png)

      </TabItem>
    </Tabs>

6. Iterate over the dataset.

    ```python
    import lavender_data.client as lavender

    lavender.init(api_url="http://0.0.0.0:8000", api_key="la-...")

    iteration = lavender.LavenderDataLoader(
        dataset_name=dataset_name,
        shuffle=True,
        shuffle_block_size=10,
    )

    for i in iteration:
        print(i["uid"])
    ```

    <LinkCard title="Learn more about LavenderDataLoader" href="/dataloader" />

</Steps>

You've successfully set up Lavender Data, created a dataset, synced data, and iterated over it!
