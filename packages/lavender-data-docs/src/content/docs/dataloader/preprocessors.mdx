---
title: LavenderDataLoader - Preprocessors
description: Learn about LavenderDataLoader features
---

Preprocessors transform batches before they're returned to your application.

```python
dataloader = LavenderDataLoader(
    dataset_id=dataset.id,
    shardsets=[shardset.id],
    preprocessors=["append_new_column"],
    batch_size=10,
)
```
